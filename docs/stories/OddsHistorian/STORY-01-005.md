## User Story

**As a** modelo de treinamento (Poisson)  
**I want to** obter uma lista de partidas finalizadas acompanhada da última odd registrada antes do início do jogo  
**So that** eu possa comparar o resultado real com a probabilidade implícita do mercado.

## Description

Para o backtesting e treinamento (STORY-02-002), não basta ter o resultado. Precisamos saber qual era a "Closing Odd" (ou a última odd capturada). Esta story envolve uma query SQL complexa que deve:
1. Filtrar partidas com status 'FINISHED'.
2. Juntar com a tabela de snapshots.
3. Pegar apenas o snapshot mais recente para cada partida cujo timestamp seja menor ou igual ao horário de início do jogo.

class OddsRepository:
    def get_finished_matches_with_closing_odds(self):
        query = """
        SELECT m.*, o.odds_home, o.odds_draw, o.odds_away
        FROM matches m
        JOIN odds_snapshot o ON o.match_id = m.match_id
        WHERE m.status = 'FINISHED'
        AND o.timestamp = (
            SELECT MAX(timestamp) FROM odds_snapshot 
            WHERE match_id = m.match_id
        )
        """
