## User Story

**As a** investidor cauteloso  
**I want to** que um alerta só seja gerado se o Modelo de Poisson E o Benchmark da Pinnacle concordarem que há valor  
**So that** eu reduza o número de falsos positivos.

## Description

Implementar a lógica híbrida (ADR-003). O sistema deve avaliar a aposta através dos dois prismas:
1. Poisson EV > 0?
2. Pinnacle Fair Odds < Soft Odds?
Se ambos forem verdadeiros (modo 'strict consensus'), o sinal é considerado de alta confiança.
