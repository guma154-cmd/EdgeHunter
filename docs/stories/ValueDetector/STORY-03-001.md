## User Story

**As a** motor de detecção de valor  
**I want to** calcular o Expected Value (EV) de uma aposta  
**So that** eu possa filtrar apenas oportunidades matematicamente lucrativas no longo prazo.

## Description

O cálculo do EV é a base do Value Betting:
`EV = (Probabilidade do Modelo * Odd da Casa) - 1`
Se EV > 0, a aposta tem valor teórico. Esta story implementa a função utilitária que realiza esse cálculo para os três mercados (1, x, 2).

## Technical Details

### Code Structure
```python
# backend/app/logic/value_calculator.py
def calculate_ev(model_prob: float, bookie_odd: float) -> float:
    return (model_prob * bookie_odd) - 1.0
```
