## User Story

**As a** investidor  
**I want to** ver o Retorno sobre o Investimento (ROI) consolidado das minhas apostas simuladas  
**So that** eu possa avaliar o desempenho real do sistema no tempo.

## Description

O ROI é a métrica principal de sucesso.
`ROI = (Lucro ou Prejuízo Total / Valor Total Apostado) * 100`
Esta story implementa o serviço que varre o histórico de apostas (pode ser simulado no início) e gera essa métrica.
