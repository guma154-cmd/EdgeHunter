## User Story

**As a** sistema de detecção  
**I want to** enviar os detalhes de uma oportunidade para o Gemini 2.0 Flash  
**So that** a IA possa realizar uma análise qualitativa (ex: notícias, contexto) e validar o sinal.

## Description

Esta story implementa a função principal de integração com o LLM (ADR-005). O sistema deve construir um prompt contendo os dados da partida (times, liga, odds, EV detectado) e perguntar à IA se a oportunidade faz sentido ou se há algum fator externo óbvio (lesão de última hora, jogo festivo) que o modelo de Poisson não capturou.

### Code Structure
```python
# backend/app/services/ai_validator.py
class AIValidator:
    def validate_opportunity(self, opportunity_data):
        prompt = self._build_prompt(opportunity_data)
        response = self.client.generate(prompt)
        return self._parse_response(response)
```

## Acceptance Criteria

[ ] Integração funcional com a API do Gemini via SDK oficial (`google-generativeai`).
[ ] Prompt estruturado para minimizar alucinações.
[ ] Função retorna um objeto com `is_valid` (bool) e `reason` (string).
[ ] O cliente Gemini possui timeout configurado de no máximo 3 segundos por chamada.

[ ] Manejo de erros de API (timeout, cota esgotada).
