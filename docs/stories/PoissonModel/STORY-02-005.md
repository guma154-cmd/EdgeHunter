## User Story

**As a** analista de qualidade  
**I want to** rodar o modelo contra dados históricos que ele não "viu" e comparar as predições com os resultados reais  
**So that** eu possa medir a Brier Score e a acurácia do modelo.

## Description

Implementar um script de backtesting. Ele deve separar os dados em Treino e Teste (Walk-forward validation é preferível em esportes). Para cada jogo no set de teste, o modelo faz a predição e comparamos com o resultado real.
Métricas:
- Brier Score (quão "calibrado" o modelo está).
- Accuracy (porcentagem de acertos do mercado mais provável).
