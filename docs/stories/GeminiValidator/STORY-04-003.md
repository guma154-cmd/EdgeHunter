## User Story

**As a** administrador do sistema  
**I want to** que o Gemini analise o fluxo de dados em busca de anomalias (ex: odds impossíveis, erros de scraping)  
**So that** eu possa filtrar lixo antes de chegar ao usuário final.

## Description

Usar o LLM como um filtro de sanidade de dados. Se o scraper reportar uma odd de 100.0 para o Real Madrid contra um time da 3ª divisão, a IA deve detectar que isso é provavelmente um erro de parsing ou de dados da fonte, marcando como anomalia.
