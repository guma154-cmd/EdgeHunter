## User Story

**As a** usuário final  
**I want to** receber alertas formatados no meu Telegram  
**So that** eu possa agir rapidamente sobre as oportunidades detectadas.

## Description

Esta story envolve a integração com a API do Telegram Bot. O alerta deve ser visualmente atraente e conter:
- Partida e Liga.
- Mercado e Odd.
- EV e Confiança (Poisson + Pinnacle).
- Link rápido (se possível).
- Botão de "Simular Aposta".
