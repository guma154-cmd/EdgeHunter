## User Story

**As a** gestor de banca  
**I want to** calcular o tamanho ideal da aposta usando o Critério de Kelly (fracionário 1/4)  
**So that** eu maximize o crescimento da banca e minimize o risco de ruína.

## Description

Implementar a fórmula de Kelly:
`% da Banca = (Probabilidade * Odd - 1) / (Odd - 1)`
Usaremos a variante "Fractional Kelly" (1/4 ou 0.25 do valor sugerido) por ser mais conservadora e adequada para apostas esportivas onde as probabilidades são estimativas.

### Code Structure

```python
# backend/app/logic/bankroll_manager.py
def calculate_kelly_stake(prob, odd, fraction=0.25):
    full_kelly = (prob * odd - 1) / (odd - 1)
    return max(0, full_kelly * fraction)
```
