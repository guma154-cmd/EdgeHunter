## User Story

**As a** sistema resiliente  
**I want to** tentar novamente uma chamada de API caso ocorra um erro temporário (ex: 503, Rate Limit)  
**So that** a disponibilidade da validação seja maximizada.

## Description

Implementar uma estratégia de retry para as chamadas ao Gemini. Usaremos a técnica de Exponential Backoff para não sobrecarregar a API durante períodos de instabilidade ou limites de cota.
