# Runbooks

Este diretório contém os guias operacionais do EdgeHunter de acordo com o padrão BMAD.

## O que vai aqui?
Documentos que descrevem procedimentos passo a passo para:
- Instalação e Configuração (Setup)
- Deploy em Produção
- Troubleshooting de Scrapers
- Procedimentos de Backup e Restore do Banco de Dados (ex: `db_restore.md`)
- Migrações e Atualizações
