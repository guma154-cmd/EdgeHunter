**So that** eu possa estimar os parâmetros de força de ataque e defesa de cada time.

## Description

Esta é a story central do módulo de predição. O modelo de Poisson clássico (Dixon-Coles simplificado) assume que o número de gols marcados por um time segue uma distribuição de Poisson com média λ.
λ_home = attack_home * defense_away * avg_home_goals
λ_away = attack_away * defense_home * avg_away_goals

Usaremos a biblioteca `scipy.optimize` para encontrar os parâmetros que maximizam a verossimilhança dos resultados históricos.
