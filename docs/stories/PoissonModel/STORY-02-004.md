## User Story

**As a** usuário do sistema  
**I want to** converter os lambdas gerados pelo modelo em probabilidades percentuais de Vitória, Empate e Derrota  
**So that** eu possa comparar com as odds das casas de apostas.

## Description

Com os lambdas (média de gols esperada) para cada time, usamos a distribuição de Poisson para calcular a probabilidade de cada placar exato (ex: 1-0, 2-1, 0-0). A soma das probabilidades dos placares que resultam em vitória do mandante dá a probabilidade 'Home'. O cálculo deve ser feito até um limite razoável de gols (ex: 10 gols por time) para garantir que a soma das probabilidades seja próxima de 100%.
